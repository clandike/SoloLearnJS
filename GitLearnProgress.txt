		Git commands
	git init - створити файл .git
	git clone <url> - створити клон репозиторію
	git remote add origin <url> - створює remote(віддалений) під назвою "origin"
	git commite -m "name" - створює коміт з назвою "name"
	git log - показує створені коміти
	git push - відправляє створений коміт на сервер
	git status - показує чи були якісь зміни в якихось файлах
	git pull - бере інформацію з сервера про оновлення файлів і оновлює їх
